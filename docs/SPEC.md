# SPEC.md
DeepSAD 모델 학습 및 검증 UI의 기능을 명시함

## 의존성 패키지
- 모든 UI는 Gradio로 구현

## 앱 실행 방식
- Gradio로 구현한 UI의 각 버튼은 함수를 호출하는 역할만 수행함

## UI의 구성
- UI는 하위에는 아래 네 개의 탭으로 구성됨
    - 학습 설정
        - 데이터셋 경로, 하이퍼 파라미터, 학습 옵션을 설정하고 train.yaml을 생성
        - 학습 시작을 누르면 생성한 train.yaml으로 학습 코드(train.py)를 호출
    - 학습 진행상황 로깅 터미널
        - 학습 코드에서 터미널에 출력하는 내용을 출력
    - 모델 검증
        - 검증 옵션을 설정하고 test.yaml을 생성
        - 검증 시작을 누르면 생성한 test.yaml으로 검증 코드(test.py)를 호출
        - 측정한 Score를 표로 화면에 출력
        - 사용자가 지정한 결과 저장 경로 하위에 저장된 FN, FP 이미지를 불러와 옆으로 넘겨가며 볼 수 있도록 함
    - 모델 테스트
        - 이미지나 이미지 폴더, 결과를 저장할 경로, 그리고 모델 pt 파일을 설정
        - 설정한 이미지로 모델 추론 수행 후 정상/이상 여부 판단
        - 각 이미지 위에 측정한 Distance, Score, Threshold 및 정상/이상 판단 여부를 그린 이미지를 저장하고, 이 이미지를 불러와 UI에서 시각화
